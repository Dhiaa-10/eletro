# Eletro ⚡

تطبيق متكامل لإدارة عدادات الكهرباء في المجمعات السكنية، يتيح مراقبة الاستهلاك، تتبع الفواتير، وإدارة بيانات الشقق والمستأجرين بكل سهولة وأمان.

A comprehensive Flutter application for managing electricity meters in residential complexes. It enables consumption monitoring, billing history tracking, and apartment management with secure biometric authentication.

## ✨ المميزات | Features

*   **📊 مراقبة الاستهلاك (Consumption Monitoring):** تتبع دقيق لاستهلاك الكهرباء لكل شقة.
*   **📑 سجل الفواتير (Billing History):** عرض الفواتير السابقة وتحليلها.
*   **🏢 إدارة الشقق (Apartment Management):** إضافة وتعديل بيانات الشقق والمستأجرين والعدادات.
*   **🔐 الأمان (Security):** دعم تسجيل الدخول عبر البصمة (Biometric Authentication).
*   **📈 رسوم بيانية (Smart Analytics):** إحصائيات بصرية توضح توجهات الاستهلاك.
*   **📱 واجهة مستخدم عصرية (Modern UI):** تصميم أنيق وتجربة مستخدم سلسة.

## 🛠 التقنيات المستخدمة | Tech Stack

*   **Framework:** Flutter
*   **State Management:** GetX
*   **Database:** SQLite (sqflite)
*   **Animations:** Flutter Animate
*   **Charts:** FL Chart
*   **Local Auth:** Fingerprint & FaceID support

## 🚀 التشغيل | Getting Started

1.  تأكد من تثبيت Flutter SDK.
2.  قم بتحميل الأكواد: `git clone https://github.com/Dhiaa-10/eletro.git`
3.  قم بتثبيت المكتبات: `flutter pub get`
4.  قم بتشغيل التطبيق: `flutter run`
